
# TODO APP

This is a Todo Application made with react. It designed to help users organize tasks, manage deadlines, and increase productivity. Typically, it allows users to create lists of tasks and mark tasks as completed. The interface is usually user-friendly, allowing for easy navigation and quick entry and editing of tasks.


## Features

- User can Create List of Tasks
- User can update the Task
- Mark task as completed
- Get List of Completed Tasks and time of completion
- Delete Task

## Running the application

  - Download the source code or clone the repository
  - cd into the directory containing the source code from 
    terminal.
  - Install dependencies using 'npm install'
  - Run the application using 'npm start'
